HINT IMAGES
===========

Drop your hint pictures in this "images" folder using these exact file names.
Each one shows up as the hint when that day's present is opened.

  Wednesday  (Diesel Hoodie)     ->  hint-wednesday.jpg
  Thursday   (Raffaello)         ->  hint-thursday.jpg
  Friday     (1x DPA)            ->  hint-friday.jpg
  Saturday   (Disneyland Paris)  ->  hint-saturday.jpg
  Sunday     (Nice Lunch)        ->  hint-sunday.jpg

Tips
----
- You can use .jpg, .jpeg, .png or .heic photos straight from your iPhone,
  but rename them to match the names above (keep the .jpg ending the simplest).
- If a file is missing, the present still opens and shows a friendly note
  telling you which file name to add.
- Portrait phone photos look great. Anything roughly square or vertical works.
